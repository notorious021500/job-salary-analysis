# job-salary-analysis
SQL + Python + Power BI project analyzing 250K job dataset

I analyzed 250K+ job records to uncover salary trends across roles, industries, and experience levels.

🔧 Tools Used:
- SQL (MySQL)
- Python (Pandas, Numpy, Seaboarn, Matplotlib)
- Power BI

📈 Key Insights:
- Remote jobs offer higher salaries
- Experience significantly impacts salary growth
- Certifications boost fresher salaries
- Anamoly found as experience dose not match salary gained
- Certain industries offer significantly higher compensation
- Certifications have a noticeable impact on fresher salaries
- Entry-level roles show less variation in salary, while higher experience levels have a wider salary range
- Salaries show a clear upward trend with experience, with a significant jump after mid-level roles (5+ years)

This project analyzes over 250K job records to understand salary trends across roles, industries, and experience levels. The analysis was initially performed using SQL to extract insights through aggregations, window functions, and anomaly detection techniques. Further exploration was done in Python using libraries like pandas, numpy, seaborn, and matplotlib to identify patterns, visualize distributions, and detect outliers that could indicate potential irregularities or fraud. The final insights were presented through an interactive Power BI dashboard, enabling dynamic analysis of salary distribution, skills impact, and experience-based growth
